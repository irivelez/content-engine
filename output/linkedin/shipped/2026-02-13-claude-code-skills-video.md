Acabo de crear un sistema que lee más de 100 noticias de IA al día y me dice las que importan.

No es una app.
No escribí código.
Es un archivo de texto con instrucciones.

Se llama un skill en Claude Code — y lo construí en menos de una hora.

Así fue el proceso:

1→ Investigué en Perplexity las fuentes más confiables de IA
   (blogs, newsletters, podcasts de las voces que importan)

2→ Convertí esas fuentes en un archivo JSON que Claude puede leer

3→ Le pedí a Claude Code que creara un skill usando este prompt:

"Create a skill in this project to help me keep updated with the most trusted information sources about AI. Only use the trusted sources referenced in the data folder to discover the most relevant signals for a business leader or entrepreneur. The outcome must be a digest with the most relevant topics, key stats and key highlights. The time frame for the news must be maximum 3 days old. Keep direct links to the sources and create a clean and attractive digest. Think like your goal is keeping C-level execs fully briefed."

El resultado: un briefing diario con lo que realmente importa.

Hoy me enteré que Anthropic tiene una valoración de $380 billion, que Google lanzó Gemini 3 Deep Think, y que OpenAI está enfrentando presión financiera seria.

Todo en 2 minutos. Sin abrir Twitter. Sin revisar 15 newsletters.

Publiqué la guía paso a paso para que crees tu propio skill — sea de IA, finanzas, tu industria, lo que necesites.
