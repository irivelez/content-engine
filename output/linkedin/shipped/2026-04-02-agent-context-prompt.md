Ahora con los agentes personales, es importante que sepan y entiendan quién eres.

Si es la primera vez que estás configurando tu agente, el punto de partida es colectar la información de los LLMs con los que has venido trabajando: ChatGPT, Gemini, Claude, etc.

En mi caso lo que hice:

1. Corrí este prompt en un chat de Claude Desktop
2. Corrí el mismo prompt en un chat de Claude Code
3. Claude Code fue más completo, porque la mayoría del tiempo estoy ahí
4. Le pedí a Claude Desktop que mezclara lo más relevante de ambos en un solo archivo
5. Como resultado: un documento con "¿Quién soy yo?" desde los ojos del agente

Usas ese documento para alimentar la configuración de tu agente.
Así es como haces que te conozca mucho mejor desde la primera vez.
