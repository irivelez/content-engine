## Key Rules

### 1. Never Copy - Always Adapt
These techniques are PATTERNS, not scripts. The words must be yours.

**Wrong**: Copy the exact hook from a 700K creator
**Right**: Understand WHY the hook works, then create YOUR version

### 2. One Experiment at a Time
Don't combine 5 new techniques in one post. You won't know what worked.

**Experiment approach**:
- Week 1: Try Specific Number Hook on 2 posts
- Track performance vs baseline
- If works → add to your proven formulas
- If doesn't → try different adaptation or different technique

### 3. Your Voice is Non-Negotiable
These techniques must work WITH your voice, not replace it.

**Always check against**:
- voice-and-style.md (does this sound like ME?)
- core-values.md (does this align with what I stand for?)
- Builder vs Philosopher (am I doing/building, not philosophizing?)

### 4. Track Everything
When you experiment with a technique, note:
- Which technique/framework you used
- How you adapted it
- Performance result (impressions, engagement)
- Why you think it worked/didn't work

Update `performance-analysis.md` when you find winning adaptations.
